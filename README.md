# 🐼 Vaccine Panda

**Due List & Quotation Maker** for vaccination clinics.

## Features
- Password-protected login (30-day session)
- Vaccine search & category filters
- Quotation & Due List modes
- WhatsApp integration
- PDF download
- PWA installable on mobile

## Setup
1. Upload this repo to GitHub
2. Go to **Settings → Pages → Source: main branch / root**
3. Your app will be live at `https://yourusername.github.io/vaccine-panda`

## Change Password
Open `index.html`, find this line near the top of the `<script>` tag:
```js
const PASS = 'panda2024';
```
Change `panda2024` to your desired password and re-upload.

## Install on Mobile (PWA)
1. Open the site in Chrome/Safari on your phone
2. Tap **Share → Add to Home Screen**
3. The app works offline after first load
